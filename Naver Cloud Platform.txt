2024.05.10(금)
-------------------------------------------------------------------------------------
상품 의존성
→ 클라우드도 물리적으로 어딘가에는 존재해야 한다

리전(Region)
→ 한국 리전, 일본 리전 
→ 나라를 뜻함

존(Zone)
→ 리전에 속한 한 구역
-------------------------------------------------------------------------------------
서버
Standard - 1:4
High CPU - 1:2
High Memory - 1:8
-------------------------------------------------------------------------------------
GPU 서버
T4 - 최대 2장의 GPU 제공
V100 - 최대 4장의 GPU 제공
A100 - Bare Metal Server로만 제공
-------------------------------------------------------------------------------------
국가별 지원 서버 타입(Region)
한국 싱가폴 일본 미국 독일 홍콩
-------------------------------------------------------------------------------------
VPC(Virtual Private Cloud)
→ 클라우드상에서 논리적으로 격리된 고객 전용 네트워크 공간
  - 현재 한국 리전 민간, 금융 클라우드에 적용
  - 계정당 최대 3개의 VPC 생성 가능
